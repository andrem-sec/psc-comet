## Navigation Rules

- **Heartbeat** loads this index and `learnings.md` (main). Does not load tagged files automatically.
- **Tagged files** are loaded on demand when a specific topic is relevant to the current session.
- **Distill** compresses old entries in tagged files and merges near-duplicates.
- **Wrap-up** routes new learnings here: presents tag options, writes to the chosen file, updates the log below.

## Main vs. Tagged

Write to `learnings.md` (main) when the learning:
- Applies in every session regardless of project or task
- Is a standing rule (naming convention, commit format, response style)
- Must never be forgotten regardless of context

Write to a tagged file for everything else.
