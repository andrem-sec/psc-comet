---
name: wrap-up
description: End-of-session protocol — collect learnings, update context, commit
---

Invoke the wrap-up skill now. Run these steps in order:

1. **Append to context/learnings.md** — add any patterns, decisions, or lessons from this session. This is the longterm project journal. Never overwrite it.

2. **Overwrite context/handoff.md** — write a fresh handoff note for the next session. Include:
   - Last commit hash and one-line summary
   - What was completed this session
   - Open risks or unresolved questions
   - Exact next step — specific enough that a cold session can pick up without asking
   - Files actively in progress (if any)

3. **Update context/user.md** if goals, preferences, or active projects changed this session.

4. **Commit** with a conventional commit message if there are uncommitted changes.
