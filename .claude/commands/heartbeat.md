name: heartbeat
description: Session-start orientation — load context, surface learnings, confirm registry
---

Invoke the heartbeat skill now. Run these steps in order:

1. **Check for context/handoff.md** — if it exists, read it first and surface the last session's open risks and next step prominently.

2. **Load context/user.md** — orient to the user's role, preferences, and active projects.

3. **Load context/learnings.md** — surface any learnings relevant to the current project state.

4. **Scan git status** — note current branch, any uncommitted changes, and whether HEAD matches the commit in handoff.md (if it exists).

5. **Produce a session brief** — current branch, last commit, open risks from handoff (if any), and suggested first action.
