Invoke the inspiration-brief skill now. Confirm brand-context is loaded first. Identify the product type and run the ui-ux-pro-max design system generator for recommendations. Collect 2-5 inspiration references (at least one from awwwards, godly.website, or Dribbble). Synthesize into a brief with: named style, color direction, typography direction, visual storytelling concept, and anti-patterns for this product type. Present the draft brief and wait for explicit confirmation before any HTML or CSS is written.
